# Peliprojekti

Tästä kansiosta löytyy kaikki yhteiseen peliprototyyppiin liittyvät tiedostot.
- *development files* kansiossa tulee olemaan pelin kehitykseen käytetyt lähdetiedostot, joka eivät tule olemaan osana peliä sen julkaisussa (esim. kuva ja audio lähdetiedostot).
- *[projektin nimi]* kansiossa tulee olemaan kaikki pelin käyttämät tiedostot, jotka tulevat olemaan osana peliä sen julkaisussa (esim. koodi-tiedostot, .PNG, .JPG, .MP3, jne.)

(Kansiot tullaan luomaan kun projektista ja sen nimestä ollaan pääty yhteisymmärrykseen.)

---

**Tämä peliprojekti tulee olemaan koko ryhmän yhteistyön tulos, jossa kaikilla osallisilla on oikeudet omiin tuotoksiinsa. Pelin tai edes peliprototyypin julkaisu on aina hieno tilaisuus ja moni varmasti haluaa näyttää sitä perheelle ja kavereille, mutta älä kuitenkaan jaa lähdetiedostoja eteenpäin ilman.**
